# Security Policy

## Project Context

This project was developed as part of a university academic capstone course.
The academic period has concluded, and the project is considered feature-complete.

The repository remains public for educational and reference purposes.

## Supported Versions

The project is **not under active development**.

Only the `main` branch is preserved as a stable reference of the final academic submission.
No guarantees are made regarding ongoing maintenance, security patches, or updates.

## Reporting a Vulnerability

If you discover a potential security vulnerability, please **do not open a public issue**.

Instead, please report the issue using GitHub's private vulnerability reporting feature,
or contact the repository owner directly through GitHub.

Responsible disclosure is appreciated.  
Please note that responses or fixes are **not guaranteed**, given the academic nature and current project status.
