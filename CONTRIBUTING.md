This repository is part of a university academic capstone project and is shared publicly
for learning, experimentation, and reference purposes.

While the project is no longer under active development, contributions are welcome.

## How to Contribute

If you would like to contribute:

1. Fork this repository
2. Create a new branch from `main`
3. Make your changes or improvements
4. Commit your changes with clear and descriptive messages
5. Open a Pull Request against the `main` branch

All pull requests will be reviewed by the repository owner.

## Contribution Scope

Please note:

- Not all pull requests are guaranteed to be accepted.
- Changes related to infrastructure, security, CI/CD workflows, or deployment
  may require additional review or may be declined.
- The goal of this repository is educational clarity rather than continuous feature growth.

## Project Status

This project has completed its academic lifecycle.
As such, pull requests may be reviewed at the discretion of the repository owner
and may not receive immediate feedback.

Thank you for your understanding and for taking the time to contribute.
